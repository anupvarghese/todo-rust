# TO DO Using Rust

Simple todo list using Rust

### How to Run

- Clone
- `cargo run`

```
  Todo List:
  ---------------
  add milk


  Todo List:
  ---------------
  [ ] 1 milk
  add banana


  Todo List:
  ---------------
  [ ] 1 milk
  [ ] 2 banana
  done 1


  Todo List:
  ---------------
  [✔] 1 milk
  [ ] 2 banana
  remove 2


  Todo List:
  ---------------
  [✔] 1 milk
```
